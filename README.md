# Abiotic-Factor-Reload-XP-Macro
Simple automation tool created for Abiotic Factor.

# Required Python, PyAutoGUI , PyAutoGUI - Keyboard, Any python compiler

# Place any ranged weapon on slot 8 then press f4 to start or pause and esc to end macro
If you want to choose any other slot then just press f2 on the weapon and f3 on the unload button
I would not recomend moving camera or keyboard while script is active
